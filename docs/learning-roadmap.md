# 学習ロードマップ

詳しい進め方は [app-first-roadmap.md](./app-first-roadmap.md) を正本にします。

## この教材で学ぶこと

- AIとの要件定義
- GitHub Issueへの分解
- Codex appでの実装依頼
- localhostでの動作確認
- PRの差分確認
- test / build / CI
- レビューコメント対応
- 本番環境との違い
- 受講者への伴走方法

## レベル

| Lv | できるようになること | 証跡 |
|---:|---|---|
| 0 | 安全ルールを説明できる | 禁止事項チェック |
| 1 | 家計簿アプリの要件を作れる | 要件メモ |
