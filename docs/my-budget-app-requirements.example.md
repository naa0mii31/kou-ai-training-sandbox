# 私の家計簿アプリ要件

## アプリ名

例: 週末まとめ家計簿

## 使う人

例: 自分。週末にPCでまとめて入力する。

## 解決したい悩み

例: 月末に何に使ったかわからなくなる。

## 最初に必ずほしい機能

- 支出を入力できる
- 月合計が見える
- カテゴリ別に見える

## あるとうれしい機能

- 月予算
- 使いすぎアラート
- CSV出力

## 今回やらないこと

- 銀行連携
- クレジットカード連携
- ログイン
- 本番公開

## 完成条件

- localhostで支出を追加できる
- 一覧と合計が表示される
- PRで差分と検証結果を説明できる
