# localhost と本番環境

## localhostとは

localhostは、自分のPCの中だけでアプリを動かす場所です。

この教材では、まず `npm run dev` でアプリを起動し、`http://localhost:3000` で確認します。

## 本番環境とは

本番環境は、他の人も見られる公開された環境です。URLが公開されるため、失敗したときの影響が大きくなります。

| 項目 | localhost | 本番環境 |
|---|---|---|
| 見られる人 | 自分だけ | 他の人も見られる |
| URL | `localhost:3000` | 公開URL |
| データ | 練習用データ | 実データが入り得る |
| 失敗の影響 | 小さい | 大きい |
| 確認方法 | 手元で操作 | 公開前後に確認 |

## この教材でのゴール

初期ゴールは、本番公開ではありません。

まずは次を説明できればOKです。

- localhostは自分のPCで動く
- 本番環境は他の人に見える
- 本番に出す前はPR、テスト、レビューが必要
- 実データや個人情報は練習環境に入れない
