# 家計簿アプリ Issue Backlog

この一覧から、受講者のレベルに合うIssueを選びます。

## Lv1: まず動く

### Issue: 支出入力フォームを作る

- 目的: 支出名、金額、カテゴリ、日付を入力できるようにする
- DoD: 入力して追加ボタンを押すと一覧に表示される
- やらないこと: 保存、編集、削除、本番公開
- 確認: localhostで1件追加する

### Issue: 支出一覧を表示する

- 目的: 登録済み支出を一覧で見られるようにする
- DoD: 支出名、金額、カテゴリ、日付が表示される
- やらないこと: 検索、集計、保存
- 確認: サンプルデータが表示される

## Lv2: 家計簿らしくする

### Issue: 月別フィルタを追加する

- 目的: 指定した月の支出だけを見られるようにする
- DoD: 月を選ぶと一覧と合計が変わる
- やらないこと: 年間集計、グラフ
- 確認: 2026-06だけ表示できる

### Issue: カテゴリ別合計を表示する

- 目的: どのカテゴリに使っているか見えるようにする
- DoD: 表示中の支出をカテゴリ別に合計できる
- やらないこと: グラフ表示
- 確認: Foodなどの合計が表示される

## Lv3: 保存と編集

### Issue: localStorageに保存する

- 目的: リロードしても支出が残るようにする
- DoD: 支出を追加後、再読み込みしても表示される
- やらないこと: サーバー保存、ログイン、外部送信
- 確認: ブラウザ再読み込みで支出が残る

### Issue: 支出を編集できるようにする

- 目的: 入力ミスをあとから直せるようにする
- DoD: 既存支出のタイトル、金額、カテゴリ、日付を変更できる
- やらないこと: 一括編集
- 確認: 1件編集して一覧に反映される

## Lv4: 自分専用機能

### Issue: 月予算を設定する

- 目的: 使いすぎに気づけるようにする
- DoD: 月予算と使用率が表示される
- やらないこと: 銀行連携、通知送信
- 確認: 予算超過時に画面でわかる

### Issue: CSV出力を追加する

- 目的: 入力した支出を手元で保管できるようにする
- DoD: 表示中の支出をCSVとしてダウンロードできる
- やらないこと: 外部アップロード
- 確認: CSVファイルを開いて内容を確認する
